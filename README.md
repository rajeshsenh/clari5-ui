# clari5-ui

> Made with create-react-library

[![NPM](https://img.shields.io/npm/v/clari5-ui.svg)](https://www.npmjs.com/package/clari5-ui) [![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)

## Install

```bash
npm install --save clari5-ui
```

## Usage

```tsx
import React, { Component } from 'react'

import MyComponent from 'clari5-ui'
import 'clari5-ui/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
